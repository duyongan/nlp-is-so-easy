1.2.1.2. 源头：SGD

$\eta$：学习率

每次使用一个样本计算损失函数, 从而得到梯度, 逐步的迭代参数

1.2.1.3. 批量学习优化器

使用一个样本容易陷入局部最优点, 且迭代速度慢,容易产生震荡,批量学习正是解决这个问题的方法.

当然在线学习有另一条路子解决,且在线学习是为了参数稀疏化而生的,更适用于搜广推领域.

1.2.1.3.1. momentum

wt+1=wt−η⋅(β1⋅mt−1+(1−β1)⋅gt) w_{t+1}=w_{t}- \eta \cdot(\beta _{1}\cdot m_{t-1}+(1- \beta _{1})\cdot g_{t}) w​t+1​​=w​t​​−η⋅(β​1​​⋅m​t−1​​+(1−β​1​​)⋅g​t​​)

momentum在SGD的基础上加上了惯性, 即如与上一步方向一致,加快速度,与上一步方向不一致减慢.但当局部最优点很深时,动量也无法拯救.

1.2.1.3.2. NAG

vt=αvt−1+η∇J(Wt−αvt−1) { v _ { t } = \alpha v _ { t - 1 } + \eta\nabla J ( W _ { t } - \alpha v _ { t -1} ) } v​t​​=αv​t−1​​+η∇J(W​t​​−αv​t−1​​)

Wt+1=Wt−vt { W _ { t + 1 } = W _ { t } - v _ { t } } W​t+1​​=W​t​​−v​t​​

NAG在SGD的基础上加入了看一步走一步的机制, 即将提前计算一步梯度, 防止小球盲目跟随上一步动量,冲出最优点.

1.2.1.3.3. AdaGrad

wt+1=wt−η0∑i=1tgi2+ε⋅gt w_{t+1}=w_{t}- \frac {\eta_0} {\sqrt{\sum _{i =1}^{t}g_i^{2}+\varepsilon}}\cdot g_{t} w​t+1​​=w​t​​−​√​∑​i=1​t​​g​i​2​​+ε​​​​​η​0​​​​⋅g​t​​

$\varepsilon$：极小的数,防止分母为 0

前面的优化器的学习率都是固定的, 而AdaGrad可随着模型的收敛自适应学习率. 即更新频率高的维度学习率逐渐减缓, 更新频率低的维度维持高学习率.

1.2.1.3.4. RMSProp

wt+1=wt−η0E[g2]t+ε⋅gt w_{t+1}=w_{t}- \frac {\eta_0} {E \left[ g^{2}\right] _{t}+\varepsilon}\cdot g_{t} w​t+1​​=w​t​​−​E[g​2​​]​t​​+ε​​η​0​​​​⋅g​t​​

AdaGrad 暴力的随着梯度累计越高, 学习率越低. RMS对此做了改进,只取一个窗口下的梯度平均值.

1.2.1.3.5. AdaDelta

wt+1=wt−Δwt w_{t+1}=w_{t}- \Delta w_{t} w​t+1​​=w​t​​−Δw​t​​

Δwt=E[Δw]t−1E[g2]t+ε⋅gt \Delta w_{t}=\frac {E[\Delta w]_{t-1}} {E \left[ g^{2}\right] _{t}+\varepsilon}\cdot g_{t} Δw​t​​=​E[g​2​​]​t​​+ε​​E[Δw]​t−1​​​​⋅g​t​​

AdaGrad 和 RMSProp 都得初始化学习率,可不可以连初始化都不用呢?AdaDelta给了答案

1.2.1.4. 在线学习优化器

在线学习的优化器用于流式数据的训练，对于数据量相当庞大，且数据相对稀疏的场景尤为合适。

1.2.1.4.1. TG

简单截断法

如何训练出稀疏性的参数呢？简单的想法就是设置一个阈值，小于这个阈值直接置零。 W(t+1)=T0(W(t)−η(t)G(t),θ) W^{(t+1)}=T_{0}(W^{(t)}- \eta ^{(t)}G^{(t)}, \theta) W​(t+1)​​=T​0​​(W​(t)​​−η​(t)​​G​(t)​​,θ)

T0(viθ)={0if∣vi∣≤θviotherwise T_{0}(v_{i}\theta)= \left\{ \begin{matrix} 0 \quad if|v_{i}| \leq \theta \\ v_{i}\quad otherwise \\ \end{matrix} \right. T​0​​(v​i​​θ)=​⎩​⎨​⎧​​​0if∣v​i​​∣≤θ​v​i​​otherwise​​​

如此暴力的做法会导致一些训练不足的参数置零，于是就有了优化版本：

别被公式吓坏了，下图可更加直观的看出区别，梯度阶段法由两个参数控制稀疏度。直观的理解：参数接近0时，推波助澜至更接近0；更进一步接近0时，直接置零。这比简单截断法要顺滑一些。

wt+12=wt−ηtgtf w_{t+ \frac{1}{2}}=w_{t}- \eta _{t}g_{t}^{f} w​t+​2​​1​​​​=w​t​​−η​t​​g​t​f​​

FOBOS将此问题拆解成了两部分，在梯度下降之后，加入了一个优化问题。即控制与梯度下降得到的结果不太远的第二范数，以及增加稀疏性的第一范数。

1.2.1.4.3. RDA

wt+1=argminw{1t∑r=1t⟨Gr,w⟩+λ∣∣w∣∣1+r2t∣∣w∣∣22} w_{t + 1} = { argmin }_{w} \{ \frac { 1 } { t } \sum _ { r= 1 } ^ { t } \big \langle G ^ { r } , w \big \rangle + \lambda || w ||_1 + \frac { r } { 2 \sqrt{t} } ||w ||_2^2\} w​t+1​​=argmin​w​​{​t​​1​​∑​r=1​t​​⟨G​r​​,w⟩+λ∣∣w∣∣​1​​+​2√​t​​​​​r​​∣∣w∣∣​2​2​​}

FOBOS只是拿了本步的梯度下降结果，在在线学习这种流式学习中，一次只学习一条数据，很容易造成参数的波动，参考的更远，可以减少这种异常点波动。

这不，RDA来了，RDA优化的第一项是梯度对参数的积分平均值，第二项是第一范数（增加稀疏性），第三项第二范数约束参数不离零太远.

wt+1=argminw{1t∑r=1t⟨Gr,w⟩+λ∣∣w∣∣1+λ∣∣w∣∣22+12∑s=1tσ(s)∣∣w−wt+12∣∣2} w _ { t + 1 } = {argmin}_{w} \{ \frac { 1 } { t } \sum _ { r= 1 } ^ { t } \big \langle G ^ { r } , w \big \rangle+ \lambda || w ||_1 +\lambda || w ||_2^2 +\frac{1}{2}\sum _{s=1}^{t}\sigma ^{(s)} | | w - w _ { t+ \frac{1}{2}} | | ^ { 2 } \} w​t+1​​=argmin​w​​{​t​​1​​∑​r=1​t​​⟨G​r​​,w⟩+λ∣∣w∣∣​1​​+λ∣∣w∣∣​2​2​​+​2​​1​​∑​s=1​t​​σ​(s)​​∣∣w−w​t+​2​​1​​​​∣∣​2​​}

FTRL是RDA和FOBOS优点的合集,即又考虑与之前的参数距离,又考虑本步梯度下降的结果.
