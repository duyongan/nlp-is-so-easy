1.3.1.2. L1，L2正则的概率分布

频率派认为参数是一个常量，优化方法为最大似然MLE，优化目标为：

argmaxθP(Y,X∣θ) { argmax }_ { \theta } P ( Y , X | \theta ) argmax​θ​​P(Y,X∣θ)

如果误差服从高斯分布，则：

​
P(Y,X∣θ)=∏i=1n12πσexp{−12σ2(y−f(x;θ))2} P(Y,X| \theta)= \prod _{i=1}^{n}\frac{1}{\sqrt{2 \pi}\sigma}exp \left\{ - \frac{1}{2 \sigma ^{2}}(y-f(x; \theta))^{2}\right\} P(Y,X∣θ)=∏​i=1​n​​​√​2π​​​σ​​1​​exp{−​2σ​2​​​​1​​(y−f(x;θ))​2​​}

取负对数之后，等价于一般的最小二乘法：

argminθ∑i=1n(yi−f(xi;θ))2 argmin_\theta \sum _{i=1}^{n}(y_{i}-f(x_{i}; \theta))^{2} argmin​θ​​∑​i=1​n​​(y​i​​−f(x​i​​;θ))​2​​

贝叶斯派认为参数不是一个常量，而是一个分布：

​
θ∼f(θ) \theta \sim f ( \theta ) θ∼f(θ)

此时进行最大似然：

argmaxθP(Y,X∣θ)f(θ) argmax_\theta P(Y,X| \theta)f(\theta) argmax​θ​​P(Y,X∣θ)f(θ)

此时取负对数：

argmin[−log(P(Y,X∣θ))−log(f(θ))] { argmin }[ - \log ( P ( Y , X | \theta )) - \log ( f ( \theta ) )] argmin[−log(P(Y,X∣θ))−log(f(θ))]

等价于：

argmin∑i=1n(yi−f(xi;θ))2−log(f(θ)) { a r g m i n } \sum _ { i = 1 } ^ { n } ( y _ { i } - f ( x _ { i } ; \theta ) ) ^ { 2 } - \log ( f ( \theta ) ) argmin∑​i=1​n​​(y​i​​−f(x​i​​;θ))​2​​−log(f(θ))

相比之前多了一项，而这一项就是正则项

  1. 假设参数符合拉普拉斯分布，则对应了L1正则（Lasso回归）
  2. 假设参数符合高斯分布，则对应了L2正则（Ridge回归）

限制了参数的分布空间，即控制了模型的空间，可有效控制模型的过拟合。
